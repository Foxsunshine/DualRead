DualRead - Privacy Policy

Last updated: April 6, 2026

Overview

DualRead ("the Extension") is a browser extension that provides bilingual translation. We are committed to protecting your privacy. This policy explains what data we collect, how we use it, and your rights.

Data Collection

The Extension does not collect, store, or transmit any personal data to our servers. We do not have any servers.

Data Stored Locally

The following data is stored locally on your device using Chrome's chrome.storage.sync API:

  • User preferences: target language, translation mode, selected engine
  • API keys: Gemini API key (if provided by you)

This data syncs across your Chrome browsers if you are signed into Chrome. It is never sent to any server other than Google's Chrome sync infrastructure.

Third-Party Services

When you use the Extension to translate text, the selected text is sent to one of the following services depending on your chosen engine:

  • Google Translate (translate.googleapis.com) — governed by Google's Privacy Policy
  • Google Gemini API (generativelanguage.googleapis.com) — governed by Google's Gemini API Terms

No data is sent to any other third party.

Permissions

  • activeTab: To read selected text and inject translations into the current page
  • storage: To save your preferences and API keys locally
  • contextMenus: To provide right-click translation options
  • Host permissions: To communicate with Google Translate and Gemini APIs only

Data Security

API keys are stored locally in your browser and are only transmitted to their respective API endpoints over HTTPS. We have no access to your API keys or any translated content.

Children's Privacy

The Extension is not directed at children under 13 and does not knowingly collect data from children.

Changes to This Policy

We may update this policy from time to time. Changes will be posted on this page with an updated date.

Contact

If you have questions about this privacy policy, please open an issue at github.com/Foxsunshine/DualRead/issues.
