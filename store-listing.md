## Detailed Description

DualRead displays translations alongside original text — no popups that block your reading, no page redirects.

Features:
- Select any text to see a bilingual tooltip with original + translation
- Translate entire pages with bilingual side-by-side display
- Two translation engines: Google Translate (free, no setup) and Gemini AI (free tier, needs API key)
- Supports 11 languages: Chinese (Simplified/Traditional), English, Japanese, Korean, French, German, Spanish, Portuguese, Russian, Arabic
- Right-click context menu for quick translation
- Clean, minimal UI that doesn't disrupt page layout
- All data stored locally — no external servers, no tracking

How to use:
1. Select text on any webpage — a translation tooltip appears instantly
2. Click "Translate entire page" for full bilingual reading
3. Switch between Google Translate and Gemini in the popup settings
4. For Gemini: get a free API key at aistudio.google.com/apikey

Privacy:
- No data collection, no analytics, no tracking
- API keys stored locally in your browser only
- Text is sent only to Google Translate or Gemini API for translation
